## My Own Laravel Boilerplate

### 1. Composer Install
```
composer install
```
### 2. Database Setup

```
php artisan migrate
```

### 3. Passport Setup
```
php artisan passport:client --personal
```

### 4. NPM Setup

- install dependencies

```
npm install
```

- development
```
npm run watch-poll
```

- production
```
npm run production
```

### Features

- Basic laravel authentication
- Dashboard CRUD operation
